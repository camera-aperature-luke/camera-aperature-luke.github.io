What are Selectors and why are they important?

Remember that you can use selectors to control how the HTML content on the page is displayed. You can change the color, the text and position and display on the page.
Attribute nameExample codeMDNDescription
Type Selectorsh1 {
  font-family: serif;
}
 MDNThe CSS type selector matches elements by node name. In other words, it selects all elements of the given type within a document.
Class selectors.class {
  font-size: 20px;
}
MDN The CSS class selector matches elements based on the contents of their class attribute.
ID selectors#ID {
  width: 200px
}
MDNThe CSS ID selector matches an element based on the value of the element's id attribute. In order for the element to be selected, its id attribute must match exactly the value given in the selector.
